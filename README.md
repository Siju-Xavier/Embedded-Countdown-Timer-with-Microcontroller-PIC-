# Embedded-Countdown-Timer-with-Microcontroller-PIC-
Designed and implemented a real-time countdown timer system on a microcontroller using PIC assembly/C, integrating UART for PC terminal display, Timer peripherals for precise 1-second updates, and Change Notification (CN) interrupts for push-button inputs. The system features multiple operating states (set, run, pause, alarm), low-power idle mode, and debounced button handling for increment/decrement, start, reset, and group info display. Developed a finite state machine with interrupt-driven logic to ensure accurate timing and robust user interaction, demonstrating proficiency in embedded systems, interrupt service routines, and power-efficient design.
